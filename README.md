# Candly
과거 차트로 공부하고 내 종목으로 예측하는 주식 차트 학습 서비스


## 프로젝트 개요
### 기획 의도
연습문제, AI 피드백, 예측 기록 복기를 통해 초보부터 고급 투자자까지 실전 감각을 훈련할 수 있는 학습 플랫폼


### 개발 기간
2025.07.03. ~ 2025.07.29. (1개월)
